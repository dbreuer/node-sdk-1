# nuxt-sample-app

> ConfigCat Node SDK example

## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```

# Try it by sending GET requests

Open these urls:

* http://localhost:3000 to test your webserver or 
* http://localhost:3000/keys to get all keys of the subscription or
* http://localhost:3000/isAwesomeFeatureEnabled to get 'isAwesomeFeatureEnabled' feautreflag's value


For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
